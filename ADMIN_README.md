# 锦秋基金 - 管理后台使用说明

## 🎯 功能概述

轻量级内容管理系统，运营同事可以通过简单的 Web 界面管理网站所有内容，无需修改代码。

## 📋 可管理的内容

### 1. 关于锦秋（4个段落）
- 中文内容编辑
- 英文内容编辑

### 2. 团队成员
- 姓名
- 英文职位
- 中文职位
- 个人链接（可选）

### 3. 项目矩阵
- 项目名称（中英文）
- 项目描述（中英文）
- 项目链接

### 4. 研究与活动
- 项目名称
- 项目描述（中英文）
- 项目链接（可选）

## 🚀 使用步骤

### 1. 设置管理员密码

在项目根目录创建 `.env.local` 文件：

```bash
# .env.local
ADMIN_PASSWORD=your_secure_password_here
```

**默认密码**：`jinqiu2025`（如果未设置环境变量）

**重要**：生产环境务必修改为强密码！

### 2. 访问管理后台

本地开发：
```
http://localhost:3000/admin
```

生产环境：
```
https://jinqiucapital.com/admin
```

### 3. 登录

输入管理员密码，点击"登录"按钮。

### 4. 编辑内容

- 使用顶部的标签页切换不同的内容区域
- 在表单中直接修改文本、链接等内容
- 点击右上角的"保存"按钮保存修改
- 看到"✓ 保存成功！"提示即表示保存完成

### 5. 查看效果

刷新网站首页即可看到更新后的内容。

## 📁 数据存储位置

所有内容数据存储在以下 JSON 文件中：

```
public/
  └── data/
      ├── content.json  # 页面文本、项目、研究活动
      └── team.json     # 团队成员信息
```

## 🔐 安全说明

1. **密码保护**：管理后台使用密码验证，确保只有授权人员可以编辑内容
2. **环境变量**：密码存储在 `.env.local` 文件中，不会提交到 Git
3. **生产部署**：在 Vercel 等平台的环境变量中设置 `ADMIN_PASSWORD`

### Vercel 部署配置

1. 进入 Vercel 项目设置
2. 找到 Environment Variables
3. 添加变量：
   - Name: `ADMIN_PASSWORD`
   - Value: `你的强密码`
   - Environment: Production

## 🛠 技术架构

```
前端页面 (/)
  └── 从 JSON 文件读取数据
  └── 自动显示最新内容

管理后台 (/admin)
  └── 登录验证
  └── 可视化编辑表单
  └── 保存到 JSON 文件

API 路由 (/api/admin/content)
  └── GET: 读取数据
  └── POST: 保存数据（需要密码验证）
```

## 📝 常见问题

### Q: 保存后前台没有更新？
A: 刷新浏览器页面，清除缓存后再试。

### Q: 忘记密码怎么办？
A: 修改 `.env.local` 文件中的 `ADMIN_PASSWORD`，或联系技术人员重置。

### Q: 可以添加新的团队成员吗？
A: 目前需要在 `team.json` 文件中手动添加，后续版本会支持界面添加。

### Q: 可以删除项目或研究活动吗？
A: 目前需要在 JSON 文件中手动删除，后续版本会支持界面删除。

## 🔄 数据备份建议

```bash
# 备份命令示例

```
