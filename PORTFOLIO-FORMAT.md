# Portfolio 数据格式说明

## 数据结构

投资组合(Portfolio)现在支持以下信息：

### 1. 项目基本信息
- **项目名称**：支持中英文独立配置
  - `name.zh`: 中文名称
  - `name.en`: 英文名称
- **项目链接**：项目官网URL（可选）
  - `link`: 项目官网地址

### 2. 创始人信息（支持多个）
每个项目可以添加多个创始人，每个创始人包含：
- **创始人姓名**：支持中英文独立配置
  - `name.zh`: 中文姓名
  - `name.en`: 英文姓名
- **创始人链接**：个人主页链接（可选）
  - `link`: 个人主页地址（如LinkedIn、Twitter等）

## JSON 数据格式示例

```json
{
  "portfolio": {
    "desc": {
      "zh": "我们投资了多个前沿 AI 领域的创新公司",
      "en": "We have invested in innovative companies across various cutting-edge AI fields"
    },
    "items": [
      {
        "name": {
          "zh": "示例公司",
          "en": "Example Company"
        },
        "link": "https://example.com",
        "founders": [
          {
            "name": {
              "zh": "张三",
              "en": "Zhang San"
            },
            "link": "https://linkedin.com/in/zhangsan"
          },
          {
            "name": {
              "zh": "李四",
              "en": "Li Si"
            },
            "link": "https://twitter.com/lisi"
          }
        ]
      }
    ]
  }
}
```

## 管理后台使用

在管理后台（`/admin`）的"投资组合"标签页中：

1. **添加项目**
   - 点击"+ 添加项目"按钮
   - 填写项目的中英文名称
   - 填写项目官网链接（可选）

2. **添加创始人**
   - 在每个项目卡片中，点击"+ 添加创始人"按钮
   - 填写创始人的中英文姓名
   - 填写创始人的个人链接（可选）
   - 可以为同一个项目添加多个创始人

3. **删除操作**
   - 点击"删除项目"按钮可以删除整个项目
   - 点击创始人条目右侧的"删除"按钮可以删除单个创始人

4. **保存**
   - 完成所有编辑后，点击页面顶部的"保存所有更改"按钮

## 前端显示效果

在主页面中，投资组合将显示为：

**中文模式：**
- 示例公司 | 创始人: 张三, 李四

**英文模式：**
- Example Company | Founders: Zhang San, Li Si

项目名称和创始人姓名都可以点击跳转到对应的链接页面（如果配置了链接）。

## 语言切换特性

- 项目名称和创始人姓名会根据当前语言设置自动切换
- 中文模式显示中文名称，英文模式显示英文名称
- "创始人"标签也会根据语言自动切换（中文："创始人"，英文："Founders"）
