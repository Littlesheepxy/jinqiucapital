# Portfolio 填写格式更新总结

## 更新日期
2025-11-17

## 更新内容

### 1. 数据结构扩展

原有的投资组合数据结构仅包含项目名称和链接，现已扩展为包含：

**新增字段：**
- ✅ 项目名称（中英文独立配置）
- ✅ 项目官网链接（可选）
- ✅ **创始人信息数组**（新增）
  - 创始人姓名（中英文独立配置）
  - 创始人个人链接（可选）
  - 支持多个创始人

### 2. 管理后台更新

**位置：** `/admin` -> "投资组合" 标签页

**新功能：**

1. **项目卡片重新设计**
   - 分为两个区块：
     - 📌 项目信息区块
     - 👤 创始人信息区块
   - 更清晰的视觉层级

2. **创始人管理**
   - 每个项目可独立添加/删除创始人
   - 支持为每个创始人配置：
     - 中文姓名
     - 英文姓名
     - 个人主页链接（LinkedIn、Twitter等）

3. **操作按钮**
   - "添加项目"：添加新的投资项目
   - "删除项目"：删除整个项目
   - "添加创始人"：为当前项目添加创始人
   - "删除"（创始人）：删除单个创始人

### 3. 前端展示更新

**位置：** 主页面 `/`

**显示格式：**

**中文模式：**
```
示例公司 | 创始人: 张三, 李四
```

**英文模式：**
```
Example Company | Founders: Zhang San, Li Si
```

**特性：**
- ✅ 项目名称可点击跳转到官网
- ✅ 创始人姓名可点击跳转到个人主页
- ✅ 根据语言切换自动显示对应语言的文本
- ✅ 支持多个创始人，用逗号分隔
- ✅ 如果没有创始人信息，则不显示创始人部分

### 4. 数据格式示例

```json
{
  "name": {
    "zh": "示例公司",
    "en": "Example Company"
  },
  "link": "https://example.com",
  "founders": [
    {
      "name": {
        "zh": "张三",
        "en": "Zhang San"
      },
      "link": "https://linkedin.com/in/zhangsan"
    },
    {
      "name": {
        "zh": "李四",
        "en": "Li Si"
      },
      "link": "https://twitter.com/lisi"
    }
  ]
}
```

## 使用指南

### 管理员操作流程

1. 访问 `/admin` 并登录
2. 点击"投资组合"标签
3. 点击"+ 添加项目"创建新项目
4. 填写项目中英文名称和官网链接
5. 点击"+ 添加创始人"为项目添加创始人
6. 填写创始人中英文姓名和个人链接
7. 可以继续添加更多创始人
8. 点击页面顶部的"保存所有更改"

### 注意事项

- 所有链接都应该包含完整的URL（如 `https://example.com`）
- 创始人链接可以是任何有效的URL（LinkedIn、Twitter、个人网站等）
- 中英文名称都应该填写，以便在不同语言模式下正确显示
- 如果不需要链接，可以留空
